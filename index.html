Current Projects

I currently focus my energy on a few projects, in mobile app making and in model making.

Forest Base

This is a diorama set in the Star Wars Universe, which includes a Rebel Base bunker, a TIE fighter and an X-Wing. It relies on an ESP8266 for WiFi and processing power, based on a LoLin V2 card. This will, eventually, be displayed in my children's room.

The source is stored in a Github repository.

Predator - 40K universe

This is a Space Marine Predator of the Salamanders Chapter. It relies on a Bluno card, which is an Arduino clone by the Chinese company DFRobot. This will join the ranks of my son's growing Salamanders army.

The source is stored in the same Github repository.

Soulbound Generator

I've bought Games Workshop's latest cooperative RPG to play some epic adventures with my wife and children. I'm building an iOS app to generate characters on the fly, which is useful for that game since it's designed for short, super-epic campaigns similar to the tabletop dungeon crawler from the same company.

I bought the game from Philibertnet.com.
